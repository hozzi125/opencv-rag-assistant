# OpenCV RAG Assistant

Мінімальна реалізація RAG-системи для відповіді на технічні питання з OpenCV
на основі офіційної документаціЇ та локальноЇ LLM.

Проєкт реалізує повний RAG-пайплайн:
- підготовка тематичного корпусу документів (OpenCV);
- генерація embeddings;
- побудова векторного сховища (Chroma);
- semantic retrieval (top-k)
- генерація відповідей з RAG та без RAG, та їх порівняння.

----------------------------------------------------------------------------------------

# Cтруктура проєкту
rag_opencv/
|-- build_docs.py          # збір та парсинг документаціЇ
|-- index_chroma.py        # embeddings + Chroma
|-- rag_pipeline.py        # RAG-пайплайн та демонстрація
|-- main.py                # CLI - точка входу
|-- sources.yaml           # cписок джерел OpenCV
|-- requirements.txt       # бібліотеки 
|--README.md               # опис та інструкціЇ
|-- data/                  # папка за файлами
    |--raw/                # сирі HTML-сторінки документаціЇ OpenCV
    |--processed/          # оброблені дані: чанки, метадані, маніфест корпусу
    |--chroma/             # локальне векторне сховище Сhroma (embeddings + індекс)


NOTE: папки data/raw, data/processed та data/chroma не зберігаються в репозиторіЇ та 
генеруються автоматично відповідними командами
----------------------------------------------------------------------------------------

-----------------------------------------------------------------------------------------

Для генераціЇ відповідей використовується локальна LLM через Ollama (LLaMa 3.1).

# Вимоги

1. Python 3.10+
2. Бібліотеки (встановлюються через `pip install -r requirements.txt)
3. Ollama:
      - не є вбудованим пакетом Python, встановлюються окремо через "https://ollama.com";
      - після встановлення необхідно 1 раз завантажити модель: ollama pull llama3.1.8b;
      - перевірка: ollama list.

----------------------------------------------------------------------------------------

# Запуск проєкту
1. Активувати віртуальне середовище:
  source .venv/Scripts/activate
2. Побудова корпусу:
  python main.py build-docs
3. Побудова векторного індексу:
  python main.py build-index
4. Демонстрація RAG:
  python main.py demo
